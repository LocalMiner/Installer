# LOCALM

**Host Minecraft Servers Locally, Seamlessly.**

LOCALM is a streamlined, peer-to-peer networking app designed to let you host and manage Minecraft servers directly from your local machine. Forget about complicated port forwarding or exposing your home network to unnecessary risks. LOCALM handles the heavy lifting so you and your friends can just play.

## 🛡️ Security First

We believe that hosting a game shouldn't compromise your network. LOCALM is built from the ground up with the **Best Privacy & Security Compliance**. 

* **Secure Peer-to-Peer Routing:** Connects players directly to your local server environment through secure, encrypted tunnels.
* **Privacy Focused:** Your IP address stays hidden. We don't collect unnecessary logs or track your in-game activity.
* **Granular Access Control:** You dictate exactly who gets to join your world. 

---

## ✨ Key Features

* **One-Click Local Hosting:** Spin up a server environment on your machine without touching your router settings.
* **Lightweight & Fast:** Optimized to run quietly in the background without eating up the system resources you need for gaming.
* **Cross-Platform Compatibility:** Easily invite friends regardless of their network setup. 
* **User-Friendly Dashboard:** A clean, intuitive interface to manage your server status, connections, and security settings.

## 🚀 Getting Started

### 1. Download
Grab the latest release of LOCALM for your operating system from the [Releases page](https://localm.eu.org). 

### 2. Install & Run
Run the installer and launch the LOCALM application. 

### 3. Start Hosting
1. Launch your local Minecraft Server.
2. Open LOCALM and click **"Start Hosting"**.
3. Share the generated secure connection address with your friends!
